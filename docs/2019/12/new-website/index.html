New Website

2019 Dec 15, Timid Robot Zehta

I have a new website! It uses the Lektor static site generator and is hosted on GitHub Pages.

The source code for the website is in the TimidRobot/zehta.me repository. See the README there for additional context and links.

Theme

The theme is created using Skeleton (responsive CSS Boilerplate). I call it "Queer Deathstar".

The repository includes the non-minified custom CSS: source/assets/static/custom.css.
